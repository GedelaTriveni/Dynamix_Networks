# Dynamix Networks - Java Projects Repository

This repository contains multiple Java-based projects developed as part of assigned tasks.  
Each project demonstrates core concepts of Java programming including console-based and GUI-based application development using Swing/JavaFX.

## 📌 Task 1: Expense Tracker System

The Expense Tracker application helps users manage their financial transactions by recording income and expenses and generating reports.

### 🔹 Implementations:
- Console-Based Application
- GUI-Based Application (Swing/JavaFX)

### 🔹 Features:
- Add income and expenses
- Categorize transactions (Food, Rent, Entertainment, etc.)
- View transaction history
- Generate financial reports (Daily, Monthly, Yearly)


## 📌 Task 2: Hotel Reservation System

The Hotel Reservation System is designed to manage hotel room bookings efficiently.

### 🔹 Features:
- View available rooms
- Book reservations
- Cancel reservations
- Manage customer information
- Admin panel to add/remove rooms
- Generate booking reports


## 🛠️ Technologies Used:
- Java
- OOP Concepts
- Swing / JavaFX
- File Handling
- Console-based Programming
